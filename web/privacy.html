Privacy

Last updated: July 2026

The short version

No accounts. No cookies. No analytics. No tracking. We do not collect, store, or transmit any personal data.

Your code stays with you

Everything you type in the playground executes inside your own browser. Solutions, test runs, and results are never sent to us or to any server. After the first visit, the site works fully offline — which is only possible because nothing depends on sending your data anywhere.

Local storage

If future features let you save progress or drafts, that data will live in your browser's local storage, on your device, under your control — with an export/backup file you own. Clearing your browser data removes it. We will never be able to read it.

Hosting

glifex.dev is a static site served by GitHub Pages. We operate no servers and receive no logs. GitHub may process standard web-server request data (such as IP addresses) to serve the site, governed by the GitHub Privacy Statement.

Contact

Questions or concerns: open an issue on the GitHub repository.
